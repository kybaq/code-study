## 배열이란?

- 데이터를 메모리 공간에 연속적으로 저장하는 구조.
  - JS / Python 등을 제외한 정적 타입 언어에서는 같은 데이터 타입만 가능
  - 따라서, Cache Hit Rate 가 높다.
- 순서대로 번호를 붙였기 때문에 인덱스를 통해 원하는 데이터에 빠르게 접근이 가능 => O(1)
- 데이터 삽입 / 삭제
  - 배열의 끝 일땐 => O(1)
  - 중간에서 삽입 / 삭제할 경우 => O(N)
- 추가적으로 차지하는 메모리 양(Overhead)가 적음.
